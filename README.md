# nextgendevs

## How to clone this repo

### Install git (if not already installed)

```shell
brew install git
```

### Clone the repo!
```shell
cd ~/code
git clone https://github.com/benbooth493/nextgendevs.git
```

And lets change to the git repositories directory

```shell
cd nextgendevs
```

Now lets open VSCode in that directory

```shell
code .
```
